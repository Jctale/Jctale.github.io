Ctags -- Tool for tracing codes

動機

較大的 project 往往切割成了非常多的 folders and files,
為了方便在 linux 下 trace 大型 project 可以使用 Ctags

功能簡介

Ctags 是一個可以快速穿梭在不同 files 間的工具，功能大致上
就是事先將各個 function 或 structure 做個 tag. 便於後續搜尋查看定義.

使用(搭配 Vim)

首先，安裝 ctags

接著進入想建立 tag 的目錄, 一般便是最上層

遞迴的為所有子目錄建立 tag

便可結合 vim 查詢各 tags

或是在 vim 中進行跳躍,可以用

Ctrl-] - Jump to the tag underneath the cursor
:ts \<tag> \<RET> - Search for a particular tag
:tn - Go to the next definition for the last tag
:tp - Go to the previous definition for the last tag
:ts - List all of the definitions of the last tag
Ctrl-t - Jump back up in the tag stack
